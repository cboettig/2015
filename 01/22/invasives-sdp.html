We propose a state model consistent with ; in this case a simple Beverton-Holt model. Note that in defining \(f\) we have assumed the harvesting take place before the population recruits.

f <- function(x, h, p){ 
  x <- x - h 
  p[[1]] * x / {1 + p[[2]] * x}  
}
p <- c(a = 6, b = 0.05)

This meets Olsonâ€™s three assumptions. We must also define our cost/reward function to meet assumptions 4 & 5:

damage <- 1
profit <- function(x, h) sapply(x, function(x)  -damage * x - h / (x - h) ) # vectorized for x
